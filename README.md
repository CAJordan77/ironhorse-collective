### Deploy to Netlify

1. Push this repo to GitHub
2. Connect repo to [Netlify](https://app.netlify.com)
3. Deploy settings are pre-configured in `netlify.toml`
4. Your site will be live at: `https://ironhorsecollective.netlify.app`

### Custom Domain

1. In Netlify dashboard → Domain settings
2. Add custom domain: `ironhorsecollective.com`
3. Update DNS records as instructed
