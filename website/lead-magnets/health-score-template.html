Customer Health Score Template

A framework for building predictive churn models for your SaaS

How to Use This Template

  1. Define your dimensions. List every signal that correlates with churn in your business. Pull 12 months of historical data.
  2. Set your weights. Weight each dimension based on its predictive power. Adjust over time as you collect more data.
  3. Score each dimension. Use the Green/Yellow/Red thresholds below to assign 0-100 points per dimension.
  4. Calculate the weighted average. CHS = (Usage x 0.30) + (Support x 0.20) + (Engagement x 0.25) + (Payment x 0.10) + (Qualitative x 0.15)
  5. Define playbooks. Create automated actions for each health tier (Green = grow, Yellow = engage, Red = save).
  6. Calibrate quarterly. Tune weights and thresholds based on actual churn data.
