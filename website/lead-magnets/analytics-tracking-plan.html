Google Tag Manager Configuration

Tags Required

  • GA4 Configuration Tag - GA4 Measurement ID, all pages
  • GA4 Event Tags - One per custom event (see table above)
  • Google Ads Conversion Tracking - After domain + Google Ads setup

Triggers Required

  • All Pages - Page View trigger for GA4 config
  • Email Forms - Form submission trigger matching input[type=email] + button click
  • Contact Form - Form submission trigger matching #contact-form
  • Payment Links - Click trigger for links containing "stripe.com" or "buy.stripe.com"
  • Downloads - Click trigger for links ending in .html (lead magnet pages)
